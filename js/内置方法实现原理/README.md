15. 节流和防抖
> 节流和防抖都是用在防止事件被频繁触发，节流是指在设定的时间内事件只能执行一次，防抖是指在频繁被触发的事件停止触发后延迟设定的时间后执行一次。这两个方法功能相似但是区别在于，节流至少每隔指定的时间就保证有规律地执行该功能，而防抖只有频繁事件停止触发后才开始计算间隔时间。
